R6Maps.com Update Notes

February 2017 Update Summary of changes since October

5 new languages

  • Huge thanks to those who contributed new translations recently:
    • Simplified Chinese - Warcy Zhu
    • Japanese - Matthew Davis
    • Russian - MasterGroosha
    • Portuguese - Paulo Novelli
    • Korean - LittleM0zart
  • The total number of languages supported is now eight (English, French and German were already available)!
  • Want to help add another? This page has info on helping add a language.  :)

3 new maps

    • Skyscraper
    • Bartlett University
    • Coastline
  • Each was added to R6Maps.com within a day of their releases, or even sooner!

Accuracy improvements

  • Objective markers:
    • Added a few missing objectives (!).
    • Updated hostage locations to be accurate within a few feet.  Note: Bomb and secure locations may be less accurate, but should always be in the right room.
    • Ensured bomb location #s match rank selection screen order, and letters (A & B) match room.
  • Breakable Surfaces:
    • Some line-of-sight floors were added or removed (e.g. main floor of Consulate).
    • Added some missing breakable walls and updated others to indicate # of reinforcements required.
  • & More:
    • Added ladder locations on all maps.
    • Added some missing room names.
  • Notice something wrong? Submit an issue on Github here OR email r6maps@gmail.com. Thanks!

Miscellaneous

  • Performance improvements:
    • Surprisingly huge CPU savings by changing the loading spinner.
    • Better initial load time by packaging page assets more effeciently.
    • Load smaller version of camera screen shots on smaller devices.
  • But wait, there's more!
    • New room label styles - none/disable and small.
    • Order number shown on camera icons.
    • On newer iPhones, uses more of the screen when added to the site is added to the home screen.
    • Added a compass (in case you forget that north always points up :).
    • & More! including several defect fixes listed here.
